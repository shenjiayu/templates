# tc-radio

> Radio component for [Vue.js](https://vuejs.org/).

- [Docs](docs/index.md)
- [Demo](https://xkeshi.github.io/eks/#/components/radio)

## Getting started

### Install

```shell
npm install tc-radio
```

### Usage

```js
import Vue from 'vue';
import TCRadio from 'tc-radio';
import 'tc-radio/dist/tc-radio.css';

Vue.use(TCRadio);
```

## Browser support

- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Opera (latest)
- Edge (latest)
- Internet Explorer 10+
