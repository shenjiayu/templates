# tc-card

> Card component for [Vue.js](https://vuejs.org/).

- [Docs](docs/index.md)
- [Demo](https://xkeshi.github.io/eks/#/components/card)

## Getting started

### Install

```shell
npm install tc-card
```

### Usage

```js
import Vue from 'vue';
import TCCard from 'tc-card';
import 'tc-card/dist/tc-card.css';

Vue.use(TCCard);
```

## Browser support

- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Opera (latest)
- Edge (latest)
- Internet Explorer 10+
