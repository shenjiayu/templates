# tc-input

> Input component for [Vue.js](https://vuejs.org/).

- [Docs](docs/index.md)
- [Demo](https://xkeshi.github.io/eks/#/components/input)

## Getting started

### Install

```shell
npm install tc-input
```

### Usage

```js
import Vue from 'vue';
import TCInput from 'tc-input';
import 'tc-input/dist/tc-input.css';

Vue.use(TCInput);
```

## Browser support

- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Opera (latest)
- Edge (latest)
- Internet Explorer 10+
