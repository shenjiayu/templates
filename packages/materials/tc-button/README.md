# tc-button

> Button component for [Vue.js](https://vuejs.org/).

- [Docs](docs/index.md)
- [Demo](https://xkeshi.github.io/eks/#/components/button)

## Getting started

### Install

```shell
npm install tc-button
```

### Usage

```js
import Vue from 'vue';
import TCButton from 'tc-button';
import 'tc-button/dist/tc-button.css';

Vue.use(TCButton);
```

## Browser support

- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Opera (latest)
- Edge (latest)
- Internet Explorer 10+
