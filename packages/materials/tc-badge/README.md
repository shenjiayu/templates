# tc-badge

> Badge component for [Vue.js](https://vuejs.org/).

- [Docs](docs/index.md)
- [Demo](https://xkeshi.github.io/eks/#/components/badge)

## Getting started

### Install

```shell
npm install tc-badge
```

### Usage

```js
import Vue from 'vue';
import TCBadge from 'tc-badge';
import 'tc-badge/dist/tc-badge.css';

Vue.use(TCBadge);
```

## Browser support

- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Opera (latest)
- Edge (latest)
- Internet Explorer 10+
