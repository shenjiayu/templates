# tc-theme-default

> The default theme for [TC](https://xkeshi.github.io/eks).

- [Demo](https://xkeshi.github.io/eks)

## Getting started

### Install

```shell
npm install eks tc-theme-default
```

### Usage

```js
import Vue from 'vue';
import TC from 'eks';
import 'tc-theme-default/dist/tc-theme-default.css'; // or `import 'eks/dist/eks.css'`

Vue.use(TC);
```

## Browser support
