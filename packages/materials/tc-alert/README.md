# tc-alert

> Alert component for [Vue.js](https://vuejs.org/).

- [Docs](docs/index.md)
- [Demo](https://xkeshi.github.io/eks/#/components/alert)

## Getting started

### Install

```shell
npm install tc-alert
```

### Usage

```js
import Vue from 'vue';
import TCAlert from 'tc-alert';
import 'tc-alert/dist/tc-alert.css';

Vue.use(TCAlert);
```

## Browser support

- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Opera (latest)
- Edge (latest)
- Internet Explorer 10+
