# tc-row

> Row component for [Vue.js](https://vuejs.org/).

- [Docs](docs/index.md)
- [Demo](https://xkeshi.github.io/eks/#/components/row)

## Getting started

### Install

```shell
npm install tc-row
```

### Usage

```js
import Vue from 'vue';
import TCRow from 'tc-row';
import 'tc-row/dist/tc-row.css';

Vue.use(TCRow);
```

## Browser support

- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Opera (latest)
- Edge (latest)
- Internet Explorer 10+
