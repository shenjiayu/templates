# tc-col

> Column component for [Vue.js](https://vuejs.org/).

- [Docs](docs/index.md)
- [Demo](https://xkeshi.github.io/eks/#/components/col)

## Getting started

### Install

```shell
npm install tc-col
```

### Usage

```js
import Vue from 'vue';
import TCCol from 'tc-col';
import 'tc-col/dist/tc-col.css';

Vue.use(TCCol);
```

## Browser support

- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Opera (latest)
- Edge (latest)
- Internet Explorer 10+
