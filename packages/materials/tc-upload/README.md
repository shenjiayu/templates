# tc-upload

> Upload component for [Vue.js](https://vuejs.org/).

- [Docs](docs/index.md)
- [Demo](https://xkeshi.github.io/eks/#/components/upload)

## Getting started

### Install

```shell
npm install tc-upload
```

### Usage

```js
import Vue from 'vue';
import TCUpload from 'tc-upload';
import 'tc-upload/dist/tc-upload.css';

Vue.use(TCUpload);
```

## Browser support

- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Opera (latest)
- Edge (latest)
- Internet Explorer 10+
