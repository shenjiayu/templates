# tc-step

> Step component for [Vue.js](https://vuejs.org/).

- [Docs](docs/index.md)
- [Demo](https://xkeshi.github.io/eks/#/components/step)

## Getting started

### Install

```shell
npm install tc-step
```

### Usage

```js
import Vue from 'vue';
import TCStep from 'tc-step';
import 'tc-step/dist/tc-step.css';

Vue.use(TCStep);
```

## Browser support

- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Opera (latest)
- Edge (latest)
- Internet Explorer 10+
