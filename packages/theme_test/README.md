# eks

> A series of Vue components.

- [Docs & Demo](https://xkeshi.github.io/eks)

## Getting started

### Install

```shell
npm install eks
```

### Usage

```js
import Vue from 'vue';
import Eks from 'eks';
import 'eks/dist/eks.css';

Vue.use(Eks);
```

## Browser support

- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Opera (latest)
- Edge (latest)
- Internet Explorer 10+
